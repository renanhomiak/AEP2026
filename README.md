- Sistema ObservaAção:
Sistema desenvolvido para facilitar o registro e acompanhamento de solicitações de serviços públicos.

- Funcionalidades:

Cidadão:
- Criar solicitação
- Consultar status por protocolo
- Opção de anonimato
Gestor:
- Listar solicitações
- Buscar por protocolo
- Atualizar status com comentário obrigatório

Regras do Sistema:
- Prioridade entre 1 e 5  
- Comentário mínimo de 10 caracteres  
- Descrição obrigatória  
- Histórico com data e responsável  
- SLA definido por prioridade  

Tecnologias:
- Java  
- Programação Orientada a Objetos  
- Aplicação em terminal (CLI)
